# 🌍 WanderLust – Travel Listing Web App

🌍 WanderLust
WanderLust is a full-stack web application inspired by Airbnb, where users can explore, create, and review travel listings. It provides a seamless experience for discovering destinations and sharing travel experiences.

---

## 🚀 Live Demo

🔗 Click Here: 👉 https://wanderlust-mern-gmmo.onrender.com

---

## 📌 About the Project

WanderLust is a dynamic web application where:

* Users can browse travel listings across different locations
* Registered users can create, edit, and delete listings
* Users can add reviews and ratings
* Only the owner of a listing/review can modify or delete it

---

## 🚀 Features

* 🔐 User Authentication (Signup/Login/Logout)
* 🏡 Create, Edit & Delete Listings
* ⭐ Add & Delete Reviews
* 🖼️ Image Upload using Cloudinary
* 💬 Flash Messages for feedback
* 👤 Authorization (Owner-based access control)
* 🌐 Fully deployed with cloud database

---

## 🛠️ Technologies Used

* **Frontend:** HTML, CSS, Bootstrap, EJS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB Atlas
* **Others:** Mongoose, Passport.js, Cloudinary, Multer, Connect-Mongo

## ▶️ How to Use

1. Sign up or log in
2. Explore available listings
3. Create your own listing
4. Add reviews and ratings
5. Edit/Delete your listings or reviews

---

## 🧠 Project Logic

* Listings and reviews are stored in MongoDB
* Authentication handled using Passport.js
* Sessions stored using MongoDB (connect-mongo)
* Authorization ensures only owners can modify content
* Flash messages provide user feedback after actions
